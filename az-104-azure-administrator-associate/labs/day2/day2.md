# Day 2 Overview

## AZ-104 Objective Alignment
- Deploy and manage Azure compute resources: Linux VM baseline deployment and administration checks.
- Implement and manage virtual networking: VNet/subnet planning, NSG segmentation, and load balancing fundamentals.
- Monitor and maintain Azure resources: connectivity validation and troubleshooting details collection.

## Lab Count
- Total labs: 5

## Labs and Estimated Time
1. [Lab 1](lab1.md) - VM Baseline Administration
- Accomplishes: deploys a cost-optimized VM baseline and validates operational readiness.
- Estimated time (new to Azure): 55-80 minutes.

2. [Lab 2](lab2.md) - VNet and NSG Segmentation
- Accomplishes: builds app/db subnet boundaries with least-privilege NSG rules and verifies segmentation intent.
- Estimated time (new to Azure): 65-90 minutes.

3. [Lab 3](lab3.md) - Connectivity and Load Balancing Starter
- Accomplishes: configures a public load balancer with health probe and backend pool validation.
- Estimated time (new to Azure): 70-95 minutes.

4. [Lab 4](lab4.md) - Compute and App Platform Gap Closure
- Accomplishes: validates VM scale set and App Service baselines for compute/platform objective closure.
- Estimated time (new to Azure): 60-90 minutes.

5. [Lab 5](lab5.md) - Networking and Operations Gap Closure
- Accomplishes: validates path-control constructs and diagnostic tooling for networking objective closure.
- Estimated time (new to Azure): 55-85 minutes.
