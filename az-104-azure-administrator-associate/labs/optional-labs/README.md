# Optional Labs Index

## Purpose
- This folder contains optional practice labs outside the 5-day core AZ-104 path.
- Use these labs for targeted reinforcement of weak areas.

## Optional Lab Count
- Total optional labs: 8

## Labs and Estimated Time
1. [Hub-and-Spoke Network Design](lab-hub-and-spoke.md)
- Accomplishes: practices hub-and-spoke VNet layout and peering validation.
- Estimated time (new to Azure): 45-70 minutes.

2. [ExpressRoute Design (Concept + Details)](lab-expressroute-design.md)
- Accomplishes: documents provider/circuit design choices and ExpressRoute vs VPN reasoning.
- Estimated time (new to Azure): 35-55 minutes.

3. [SSPR Policy Review (Entra)](lab-sspr-review.md)
- Accomplishes: reviews SSPR scope/registration settings and captures tenant constraints.
- Estimated time (new to Azure): 20-35 minutes.

4. [Storage Access and Security Gaps](lab-storage-access-security-gaps.md)
- Accomplishes: reinforces storage access controls, SAS/key workflow, and network restrictions.
- Estimated time (new to Azure): 40-65 minutes.

5. [Compute and App Platform Gaps](lab-compute-app-platform-gaps.md)
- Accomplishes: reinforces compute and managed platform administration paths.
- Estimated time (new to Azure): 45-70 minutes.

6. [Advanced Networking Gaps](lab-advanced-networking-gaps.md)
- Accomplishes: reinforces endpoint, DNS, route, and advanced network control reasoning.
- Estimated time (new to Azure): 50-75 minutes.

7. [Monitor and Backup/DR Gaps](lab-monitor-backup-dr-gaps.md)
- Accomplishes: reinforces monitoring diagnostics and backup/recovery details capture.
- Estimated time (new to Azure): 45-70 minutes.

8. [Private DNS Zone and VNet Link (Auto-Registration)](lab-private-dns-zone-vnet-link.md)
- Accomplishes: reinforces private DNS zone creation, VNet linking, and registration-enabled design choices.
- Estimated time (new to Azure): 35-55 minutes.
