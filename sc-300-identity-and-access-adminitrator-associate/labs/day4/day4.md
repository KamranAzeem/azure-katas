# Day 4 Overview

## SC-300 Objective Alignment
- Plan and implement identity governance.
- Implement Privileged Identity Management (PIM) and access reviews.
- Implement entitlement management and lifecycle workflows.

**Note:** Some Day 4 capabilities require Microsoft Entra ID P2.

## Lab Count
- Total labs: 4

## Labs and Estimated Time
1. [Lab 1](lab1.md) - PIM Role Eligibility Baseline
- Accomplishes: configures role eligibility and activation controls for privileged access.
- Estimated time: 50-75 minutes.

2. [Lab 2](lab2.md) - Access Reviews Baseline
- Accomplishes: creates access review for a critical group and validates reviewer flow.
- Estimated time: 40-65 minutes.

3. [Lab 3](lab3.md) - Entitlement Management Access Package Baseline
- Accomplishes: creates catalog/package/policy and validates request lifecycle.
- Estimated time: 50-80 minutes.

4. [Lab 4](lab4.md) - Lifecycle Workflows Baseline
- Accomplishes: builds joiner/mover/leaver flow and verifies execution behavior.
- Estimated time: 45-70 minutes.
