# Day 3 Overview

## SC-300 Objective Alignment
- Plan and implement workload identities.
- Manage app registrations, service principals, and enterprise app access.
- Implement managed identities and app credentials security practices.

**Note:** Day 3 uses both Entra admin center and Azure Portal.

## Lab Count
- Total labs: 4

## Labs and Estimated Time
1. [Lab 1](lab1.md) - App Registration and Enterprise Application Baseline
- Accomplishes: creates app registration, validates service principal, and configures ownership.
- Estimated time: 45-65 minutes.

2. [Lab 2](lab2.md) - App Roles and API Permissions
- Accomplishes: configures delegated/app permissions and consent workflow.
- Estimated time: 45-70 minutes.

3. [Lab 3](lab3.md) - Managed Identity and Key Vault Access Pattern
- Accomplishes: enables managed identity and verifies secret access path.
- Estimated time: 50-75 minutes.

4. [Lab 4](lab4.md) - Service Principal Credential Hygiene
- Accomplishes: rotates credentials and validates expiration/least-privilege posture.
- Estimated time: 40-60 minutes.
